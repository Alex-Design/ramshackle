About Ramshackle

Who should use this thing?

People new to web development that don't know any better. Or people who want a simple but effective responsive boilerplate with some light styling.

Why do you use a base-10 grid when all of the cool kids use 12?

How about I just make mine base-100 and then I'll be 8.33333 times cooler than they are?

Why did you build this?

I'm a wannabe front-end developer that wanted a good project to improve my coding skills, learn responsive design, to have a nice project to add to my portfolio, and a system that I can use to build other sites. That's like, 4 birds with 1 stone!

Why do you ask so many questions?

Because I need some content to fluff up this section of the site.

Responsive Grid

This grid uses a base-10 system with a max width of 1000px. You can add columns in increments of 10 all the way up to 100. It's like the metric system of responsive grids, but also allows for some in-between sizes like 1/4, 1/3, 2/3, and 3/4. The max width can be changed with a single number, and all content should scale proportionally. Ramshackle also uses 2 different media query breakpoints for tablets and smart phones so your content will look great on any device.

Go ahead and resize the browser, and take it for a spin.

The Syntax



                <div class="grid-container">

                <div class="row">

                <div class="col four">Four</div>

                <div class="col six">Six</div>

                </div>

                <div class="row">

                <div class="col quarter">1/4</div>

                <div class="col three-quarters">3/4</div>

                </div>

                </div>

.grid-container
Acts as the centered wrapper with a max width of 1000px

.row
All columns should be the immediate children of a row.

.col
Add 'col' and a number and you're all set.

Additional Features

.hero
This is for adding full flood background images like you see at the top of this page.

.main-container
Use this if you'd like a full flood background for your grid.

.col-row
Use this if you'd like to convert a column into a row. You can then place multiple columns inside.
Please note that using this will cause your margins to scale.

.flush
Ramshackle adds in padding for each column by default. If you'd like to remove that and tighten things up, add this class to a row, column, or to any other element.

.v-25 and .v-50
This will add 25px or 50px of vertical margin for when an element needs some room to breathe.

.h-25 and .h-50
This will add 25px or 50px of horizontal margin for when an element needs some room to stretch.

.left .right and .center
When applied to a row it will effect the rows horizontally. When applied to a column it will align the content within the column. These can also be applied to other elements as well.
